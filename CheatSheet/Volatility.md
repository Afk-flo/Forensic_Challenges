# Volatility CheatSheet 
This file is designed to help you through your memory forensic investigation. 

You'll find here commands from Volatility3 and Volatility2


vol.py -> Python code for Volatility[x] (should be the same for you)

evidence.mem -> This will be your own evidence file, the name will change for you.

## Windows
### Imageinfo 
V3 : vol.py -f evidence.mem windows.info 
V2 : vol.py -f evidence.mem imageinfo
